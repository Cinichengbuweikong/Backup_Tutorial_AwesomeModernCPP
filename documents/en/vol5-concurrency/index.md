# Volume 5: Concurrent Programming

From thread primitives to asynchronous coroutines, from locks to lock-free, from synchronous to task-based — Volume 5 helps you build complete concurrency judgment. Our principle is: **correctness first, performance second; locks first, lock-free second; synchronous first, task-based second**.
