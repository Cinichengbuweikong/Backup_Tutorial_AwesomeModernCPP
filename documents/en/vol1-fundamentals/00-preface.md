# Preface

Hey there! Welcome to the Modern C++ for Embedded Systems tutorial. To be precise, this tutorial takes a rather ambitious approach: it attempts to use C++, particularly modern C++, in embedded development to handle common calls across various layers of MCUs (Microcontroller Units) and Linux. The author has always wanted to explore this idea. Recently, finding the time to finally dive into this work, the plan is to write a detailed tutorial series to make it a reality.
