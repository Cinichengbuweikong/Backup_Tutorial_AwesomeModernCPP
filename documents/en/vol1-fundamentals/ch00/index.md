# Environment Setup and the First Program

Getting started is often the hardest part—but setting up the environment itself shouldn't be a roadblock. In this chapter, we will set up a development environment from scratch. Whether you are using Linux or Windows, it only takes a few steps to get ready. Then we will dive right into writing our first C++ program, see the full process of compilation and execution firsthand, and build an intuitive impression of the language.

This chapter is intended for readers with absolutely no C++ experience. If you already have a preferred toolchain, you can skip the environment setup and jump straight to the first program.
